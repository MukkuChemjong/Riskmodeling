Feature Explainability
What are SHAP Values?
SHAP values show how much each feature contributed to pushing the model's prediction toward or away from bankruptcy for each company.

Positive (+) → pushes toward Bankrupt
Negative (−) → pushes toward Legitimate
How to Read the Beeswarm
Each dot = one company.
Each row = one feature.
Red = high feature value.
Blue = low feature value.

Wide spread = high variability in how this feature impacts different companies.
